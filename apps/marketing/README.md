# Marketing App

Public Next.js site and managed-mode auth entry pages.

Runs in development through `pnpm --filter marketing dev` and in Compose on `http://localhost:3000`.

Important config: auth secrets, OAuth credentials, `NEXT_PUBLIC_API_URL`, `NEXT_PUBLIC_AUTH_ORIGIN`, `NEXT_PUBLIC_CONSOLE_URL`, and `CONTACT_EMAIL`.

Commands: `pnpm --filter marketing dev`, `check-types`, `lint`, `build`, `start`.

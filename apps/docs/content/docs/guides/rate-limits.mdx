# Rate Limits

Authenticated service requests are checked against account plan limits. The API emits:

- `X-RateLimit-Limit`
- `X-RateLimit-Remaining`
- `X-RateLimit-Reset`
- `Retry-After` on rate-limit failures
- `X-Quota-Limit`
- `X-Quota-Used`
- `X-Quota-Remaining`

Rate-limit failures return `429 RATE_LIMITED`. Monthly quota failures return `429 QUOTA_EXCEEDED`.

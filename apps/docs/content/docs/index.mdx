# Welcome to Planisfy

Planisfy is a geospatial API platform with self-hosted and managed v1 modes,
MapLibre style publishing, uploaded tilesets, API keys, usage tracking, and
operator tooling around open geospatial engines.

## Quick Start

### 1. Prepare Access

Create a Console account and, when needed, create an API key from the dashboard.
Published public map assets can be loaded anonymously; service APIs such as
geocoding and routing still require API key or session auth.

### 2. Replace Your Mapbox URL

```javascript
// Before (Mapbox)
map.setStyle('https://api.mapbox.com/styles/v1/mapbox/streets-v12?access_token=YOUR_KEY')

// After (Planisfy)
map.setStyle('https://api.planisfy.com/styles/v1/planisfy/streets?access_token=YOUR_KEY')
```

### 3. That's It!

MapLibre GL JS and standard z/x/y tile coordinates work with Planisfy's public
style and tile URLs.

## What is Planisfy?

Planisfy provides Mapbox-compatible API shapes where practical using:

- **[Martin](https://github.com/maplibre/martin)** — Vector tile server (Rust)
- **[Pelias](https://github.com/pelias/pelias)** — Geocoding engine (Node.js)
- **[Valhalla](https://github.com/valhalla/valhalla)** — Routing engine (C++)
- **[Overture Maps](https://overturemaps.org/)** — Open map data source

Uploaded published tilesets are served through Planisfy API-owned TileJSON and
tile URLs. Martin remains the static/local tile engine for configured PMTiles
sources; Valhalla requires a mounted routing graph before route APIs are ready.

| Feature | Planisfy | Mapbox |
|---------|----------|--------|
| **Self-Hostable** | ✅ Full control | ❌ Cloud-only |

## Core APIs

| API | Description | Status |
|-----|-------------|--------|
| **Vector Tiles** | Uploaded/static vector tiles and TileJSON | Available |
| **Style JSON** | MapLibre style specifications and publishing | Available |
| **Sprites & Glyphs** | Public map assets for rendering | Available |
| **Geocoding** | Provider-backed forward/reverse search | Provider dependent |
| **Directions** | Valhalla-backed routing | Graph dependent |
| **Isochrones** | Valhalla-backed time-distance polygons | Graph dependent |
| **Static Images** | PNG map snapshots | Requires `STATIC_MAP_URL` |

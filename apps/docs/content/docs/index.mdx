# Welcome to Planisfy

Planisfy is a self-hosted, Mapbox-compatible API platform built with open-source geospatial engines. Switch by changing one URL — keep the same code.

## Quick Start

### 1. Get Your API Key

Sign up at [planisfy.com](https://planisfy.com) and create an API key from the dashboard.

### 2. Replace Your Mapbox URL

```javascript
// Before (Mapbox)
map.setStyle('https://api.mapbox.com/styles/v1/mapbox/streets-v12?access_token=YOUR_KEY')

// After (Planisfy)
map.setStyle('https://api.planisfy.com/styles/v1/planisfy/streets?access_token=YOUR_KEY')
```

### 3. That's It!

Everything else works the same — MapLibre GL JS, Mapbox GL JS, tile coordinates, all of it.

## What is Planisfy?

Planisfy provides drop-in Mapbox API compatibility using:

- **[Martin](https://github.com/maplibre/martin)** — Vector tile server (Rust)
- **[Pelias](https://github.com/pelias/pelias)** — Geocoding engine (Node.js)
- **[Valhalla](https://github.com/valhalla/valhalla)** — Routing engine (C++)
- **[Overture Maps](https://overturemaps.org/)** — Open map data source

All open-source, all self-hostable, all compatible with Mapbox's API specs.

## Core APIs

| API | Description | Status |
|-----|-------------|--------|
| **Vector Tiles** | Protocol Buffer vector tiles | ✅ Available |
| **Style JSON** | MapLibre style specifications | ✅ Available |
| **Sprites & Glyphs** | Map icons and text rendering | ✅ Available |
| **Geocoding** | Forward/reverse address search | 🟡 In Development |
| **Directions** | Turn-by-turn routing | 🟡 In Development |
| **Isochrones** | Time-distance polygons | 🔜 Planned |
| **Static Images** | PNG map snapshots | 🔜 Planned |

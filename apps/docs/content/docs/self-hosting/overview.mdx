# Self-Hosting Overview

The Docker Compose stack runs the Planisfy apps, worker, database, Redis, and local geospatial engines. It is useful for development, demos, and self-host trials.

Core services: API, Console, Admin, Docs, Marketing, worker-geodata, Postgres, Redis, Martin, Valhalla, Pelias, elevation, and static-renderer.

Optional profiles: MinIO, Traefik, Pelias fixture import, and self-host supervisor.

Binary map data is not committed. Place PMTiles, fonts, Valhalla graph data, DEM files, and Pelias fixture/import data under `infra/docker/data` or configure external providers.

# Data Sources

Planisfy does not ship production map datasets.

- PMTiles: mount files under `infra/docker/data/pmtiles` for Martin or publish uploaded tilesets through Console.
- Fonts: hydrate `infra/docker/data/fonts` for Martin glyph serving.
- Valhalla: place graph/runtime data under `infra/docker/data/valhalla_data`.
- Pelias: use the bundled API/Elasticsearch for smoke coverage or run the official Pelias project for real imports.
- Elevation: place SRTM `.hgt` files under `infra/docker/data/elevation`.

# Glyphs And Fonts

Planisfy proxies glyph PBF files from `GLYPHS_URL`.

```text
GET /fonts/v1/{fontstack}/{range}.pbf
```

The API strips the `.pbf` suffix before requesting the configured glyph backend. Docker Compose points glyphs at Martin's font endpoint.

Responses use `Content-Type: application/x-protobuf` and `Cache-Control: public, max-age=86400`.

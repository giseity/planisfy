# Styles API

The **Planisfy Styles API** provides MapLibre-compatible style JSON specifications that define how map data is rendered. Our styles are drop-in compatible with Mapbox — just change the base URL and keep the same MapLibre Style Specification format.

<Callout type="info" title="Style Specification">
Planisfy styles follow the <a href="https://maplibre.org/maplibre-style-spec/" target="_blank">MapLibre Style Specification</a>, which is based on the open Mapbox Style Specification.
</Callout>

---

## Endpoint Overview

```
GET https://api.planisfy.com/styles/v1/{owner}/{style_id}?access_token=YOUR_KEY
GET https://api.planisfy.com/styles/v1/{owner}/{style_id}@{version}?access_token=YOUR_KEY
```

| Component | Description |
|-----------|-------------|
| `owner` | The account owner (`planisfy` for public styles) |
| `style_id` | The style identifier (`basic`, `streets`, `dark`, `satellite`) |
| `version` | Optional immutable published version number |
| `access_token` | Your Planisfy API key |

Stable style URLs resolve to the latest published snapshot. Version-pinned URLs
only resolve for snapshots that have actually been published, so draft saves do
not leak into public `@version` URLs.

---

## Available Styles

| Style ID | Style URL | Description | Best For |
|----------|-----------|-------------|----------|
| `basic` | `planisfy/basic` | Clean, minimalist design | Data visualization, dashboards |
| `streets` | `planisfy/streets` | Detailed street map with labels | Navigation, local search |
| `dark` | `planisfy/dark` | Dark theme with muted colors | Night mode, embedded maps |
| `satellite` | `planisfy/satellite` | Satellite imagery with labels | Terrain visualization, outdoor apps |

<Callout type="tip" title="Default Style">
If you don't specify a style, <code>basic</code> is used as the default.
</Callout>

---

## Request Parameters

### Required Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `owner` | `string` | Account owner (`planisfy` for public styles, or your username for custom styles) |
| `style_id` | `string` | Style identifier. See [Available Styles](#available-styles) |
| `access_token` | `string` | Your API key for authentication |

### Optional Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `fresh` | `boolean` | Force a fresh response, bypassing cache (`true`/`false`) |

## Authentication

Include your API key in one of two ways:

### Query Parameter (Recommended for SDKs)

```bash
GET https://api.planisfy.com/styles/v1/planisfy/basic?access_token=pk.your-key-here
```

### Bearer Token

```bash
GET https://api.planisfy.com/styles/v1/planisfy/basic
Authorization: Bearer pk.your-key-here
```

<Callout type="warn" title="Self-Hosted Mode">
When self-hosting, you can configure anonymous access or use your own authentication scheme.
</Callout>

## Example Requests

### cURL

```bash
# Fetch the basic style JSON
curl "https://api.planisfy.com/styles/v1/planisfy/basic?access_token=pk.your-key-here" \
  -H "Accept: application/json"

# Fetch immutable published version 3
curl "https://api.planisfy.com/styles/v1/planisfy/basic@3?access_token=pk.your-key-here" \
  -H "Accept: application/json"
```

### JavaScript (MapLibre GL JS)

```javascript
// Use Planisfy style directly in MapLibre GL JS
const map = new maplibregl.Map({
  container: 'map',
  style: 'https://api.planisfy.com/styles/v1/planisfy/streets?access_token=YOUR_KEY',
  center: [-74.006, 40.7128], // New York
  zoom: 12
});
```

### JavaScript (Mapbox GL JS)

```javascript
// Planisfy styles work with Mapbox GL JS too
const map = new mapboxgl.Map({
  container: 'map',
  style: 'https://api.planisfy.com/styles/v1/planisfy/dark?access_token=YOUR_KEY',
  center: [-74.006, 40.7128],
  zoom: 12
});
```

### Python

```python
import requests

url = "https://api.planisfy.com/styles/v1/planisfy/basic"
params = {"access_token": "pk.your-key-here"}

response = requests.get(url, params=params)

if response.status_code == 200:
    style_json = response.json()
    print(f"Style name: {style_json['name']}")
    print(f"Layers: {len(style_json['layers'])}")
```

### Node.js

```javascript
import fetch from 'node-fetch';

const response = await fetch(
  'https://api.planisfy.com/styles/v1/planisfy/streets?access_token=pk.your-key-here'
);
const style = await response.json();

console.log(`Style: ${style.name}`);
console.log(`Sources: ${Object.keys(style.sources).join(', ')}`);
```

## Response

The API returns a **JSON object** conforming to the MapLibre Style Specification:

```
Content-Type: application/json
Cache-Control: public, max-age=43200
Access-Control-Allow-Origin: *
```

### Style Object Structure

```json
{
  "version": 8,
  "name": "Planisfy Basic",
  "metadata": {
    "planisfy:owner": "planisfy",
    "planisfy:style:id": "basic",
    "planisfy:style:version": "1.0.0"
  },
  "sources": {
    "planisfy": {
      "type": "vector",
      "url": "https://api.planisfy.com/tiles/v1/basic.json?access_token=YOUR_KEY"
    }
  },
  "sprite": "https://api.planisfy.com/styles/v1/planisfy/basic/sprite",
  "glyphs": "https://api.planisfy.com/fonts/v1/{fontstack}/{range}.pbf",
  "layers": [
    {
      "id": "background",
      "type": "background",
      "paint": {
        "background-color": "#f8f8f8"
      }
    },
    {
      "id": "roads",
      "type": "line",
      "source": "planisfy",
      "source-layer": "roads",
      "paint": {
        "line-color": "#cccccc",
        "line-width": 1
      }
    }
    // ... more layers
  ]
}
```

### Style Object Properties

| Property | Type | Description |
|----------|------|-------------|
| `version` | `number` | Style specification version (always `8`) |
| `name` | `string` | Human-readable name for the style |
| `metadata` | `object` | Additional style information and versioning |
| `sources` | `object` | Data sources (tile URLs, GeoJSON, etc.) |
| `layers` | `array` | Rendered layers in draw order |
| `sprite` | `string` | URL template for sprite images |
| `glyphs` | `string` | URL template for font glyphs |
| `center` | `array` | Default center `[lng, lat]` |
| `zoom` | `number` | Default zoom level |
| `bearing` | `number` | Default bearing (rotation) |
| `pitch` | `number` | Default pitch (tilt) |

## Style Customization

You can customize Planisfy styles by modifying the style JSON after retrieving it:

### Adding a Custom Layer

```javascript
// Fetch the base style
const response = await fetch(
  'https://api.planisfy.com/styles/v1/planisfy/basic?access_token=YOUR_KEY'
);
const style = await response.json();

// Add a custom layer on top
style.layers.push({
  id: 'my-custom-pois',
  type: 'circle',
  source: 'planisfy',
  'source-layer': 'pois',
  filter: ['==', ['get', 'category'], 'restaurant'],
  paint: {
    'circle-radius': 8,
    'circle-color': '#FF5733'
  }
});

// Use the modified style
const map = new maplibregl.Map({
  container: 'map',
  style: style
});
```

### Changing Colors

```javascript
// Modify layer colors dynamically
map.setPaintProperty('roads', 'line-color', '#FF0000');
map.setPaintProperty('water', 'fill-color', '#0000FF');
```

### Filtering Data

```javascript
// Only show highways
map.setFilter('roads', ['==', ['get', 'class'], 'highway']);

// Show multiple road types
map.setFilter('roads', [
  'in',
  ['get', 'class'],
  'highway',
  'primary',
  'secondary'
]);
```

## Style Versioning

Planisfy published styles use immutable integer snapshot versions:

| Type | Description | Example |
|------|-------------|---------|
| **Stable** | Latest published snapshot | `planisfy/basic` |
| **Pinned** | Specific published snapshot | `planisfy/basic@3` |

<Callout type="info" title="Pinning Versions">
To pin a specific published style version, append <code>@version</code> to the style ID:<br/>
<code>planisfy/basic@3</code>
</Callout>

## Caching

Style JSON responses are cached to improve performance:

| Cache Type | Duration | Description |
|------------|----------|-------------|
| **Browser/Client** | 12 hours | Local cache in your app |
| **CDN/Edge** | 5 minutes | Cloudflare edge locations |
| **Origin** | 5 minutes | Planisfy servers |

<Callout type="tip" title="Cache Headers">
Responses include <code>Cache-Control</code> and <code>ETag</code> headers. Implement conditional requests with <code>If-None-Match</code> for bandwidth savings.
</Callout>

### Conditional Request Example

```bash
curl "https://api.planisfy.com/styles/v1/planisfy/basic?access_token=YOUR_KEY" \
  -H "If-None-Match: \"33a64df551425fcc55e4d42a148795d9f25f89d4\""
```

If the style hasn't changed, you'll receive a `304 Not Modified` response.

## Rate Limits

| Plan | Requests per Minute | Requests per Day |
|------|---------------------|------------------|
| Free | 1,000 | 50,000 |
| Starter | 10,000 | 500,000 |
| Pro | 100,000 | 5,000,000 |
| Enterprise | Custom | Custom |

Exceeding limits returns `HTTP 429` with a `Retry-After` header.

---

## Error Codes

| HTTP Code | Error | Description |
|-----------|-------|-------------|
| `400` | Bad Request | Invalid style ID or malformed request |
| `401` | Unauthorized | Missing or invalid API key |
| `403` | Forbidden | API key suspended or over quota |
| `404` | Not Found | Style does not exist |
| `429` | Too Many Requests | Rate limit exceeded |
| `500` | Server Error | Internal server error |

### Error Response Format

```json
{
  "message": "Style not found: planisfy/unknown",
  "code": "style_not_found",
  "request_id": "req_abc123xyz"
}
```

## Switching from Mapbox

Replace your Mapbox style URL:

### Mapbox GL JS

```javascript
// Before (Mapbox)
const map = new mapboxgl.Map({
  container: 'map',
  style: 'mapbox://styles/mapbox/streets-v12',
  accessToken: 'YOUR_MAPBOX_TOKEN'
});

// After (Planisfy)
const map = new mapboxgl.Map({
  container: 'map',
  style: 'https://api.planisfy.com/styles/v1/planisfy/streets?access_token=YOUR_PLANISFY_KEY'
});
```

### MapLibre GL JS

```javascript
// Before (Mapbox)
const map = new maplibregl.Map({
  container: 'map',
  style: 'https://api.mapbox.com/styles/v1/mapbox/streets-v12?access_token=YOUR_MAPBOX_TOKEN'
});

// After (Planisfy)
const map = new maplibregl.Map({
  container: 'map',
  style: 'https://api.planisfy.com/styles/v1/planisfy/streets?access_token=YOUR_PLANISFY_KEY'
});
```

<Callout type="info" title="Drop-In Replacement">
Planisfy styles are designed to be a drop-in replacement for Mapbox styles. The layer structure, source format, and property names are compatible.
</Callout>

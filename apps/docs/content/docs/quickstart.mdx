# Quick Start Guide

Get your first Planisfy map running in under 5 minutes. This guide will walk you through creating an account, getting an API key, and rendering your first interactive map.

---

## Step 1: Get Your API Key

### Sign Up

1. Go to [planisfy.com](https://planisfy.com) and create a free account
2. Check your email for a verification link
3. Log in to access your dashboard

### Create an API Key

1. In your dashboard, click **"Create API Key"**
2. Give your key a descriptive name (e.g., "Development")
3. Choose the appropriate scopes for your use case
4. Copy your new API key — it starts with `pk.`

<Callout type="warn" title="Keep Your Key Secret">
Treat your API key like a password. Never commit it to version control. Use environment variables or `.env` files in development.
</Callout>

---

## Step 2: Choose Your Library

Planisfy works with both MapLibre GL JS (recommended) and Mapbox GL JS. Choose based on your needs:

| Library | Recommended | License | Notes |
|---------|-------------|---------|-------|
| **MapLibre GL JS** | ✅ Yes | BSD-2-Clause | Open-source fork, actively maintained |
| **Mapbox GL JS** | ⚠️ Legacy | Proprietary | Original library, restricted terms |

## Step 4: Customize Your Map

Now that your map is rendering, let's customize it with common features.

### Change the Style

Planisfy offers several pre-built styles:

| Style ID | Description | Preview |
|----------|-------------|---------|
| `basic` | Clean, minimalist | <img src="https://via.placeholder.com/100x60/f0f0f0/999?text=basic" alt="basic" /> |
| `streets` | Detailed street map | <img src="https://via.placeholder.com/100x60/e0e0e0/999?text=streets" alt="streets" /> |
| `dark` | Dark theme | <img src="https://via.placeholder.com/100x60/333/fff?text=dark" alt="dark" /> |
| `satellite` | Satellite imagery | <img src="https://via.placeholder.com/100x60/228B22/fff?text=sat" alt="satellite" /> |

```javascript
// Switch to dark theme
map.setStyle('https://api.planisfy.com/styles/v1/planisfy/dark?access_token=YOUR_KEY');
```

### Add a Marker

```javascript
// Create a marker
const marker = new maplibregl.Marker()
  .setLngLat([-74.006, 40.7128]) // New York City coordinates
  .addTo(map);

// Or with a popup
const markerWithPopup = new maplibregl.Marker()
  .setLngLat([-74.006, 40.7128])
  .setPopup(new maplibregl.Popup({ offset: 25 }).setText('Hello, Planisfy!'))
  .addTo(map);
```

### Add Navigation Controls

```javascript
// Add zoom and rotation controls
map.addControl(new maplibregl.NavigationControl(), 'top-right');

// Add a scale bar
map.addControl(new maplibregl.ScaleControl({ maxWidth: 100, unit: 'metric' }), 'bottom-left');
```

### Add a Fullscreen Button

```javascript
map.addControl(new maplibregl.FullscreenControl());
```

## Complete Example

Here's a complete example with all the features we've covered:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Planisfy Map Example</title>
  <link href="https://unpkg.com/maplibre-gl@4/dist/maplibre-gl.css" rel="stylesheet" />
  <style>
    body { margin: 0; padding: 0; }
    #map { width: 100%; height: 100vh; }
  </style>
</head>
<body>
  <div id="map"></div>

  <script src="https://unpkg.com/maplibre-gl@4/dist/maplibre-gl.js"></script>
  <script>
    const map = new maplibregl.Map({
      container: 'map',
      style: 'https://api.planisfy.com/styles/v1/planisfy/streets?access_token=YOUR_KEY',
      center: [-74.006, 40.7128],
      zoom: 12
    });

    // Add navigation controls
    map.addControl(new maplibregl.NavigationControl(), 'top-right');

    // Add scale bar
    map.addControl(new maplibregl.ScaleControl({ maxWidth: 100 }), 'bottom-left');

    // Add fullscreen button
    map.addControl(new maplibregl.FullscreenControl());

    // Add marker with popup
    new maplibregl.Marker()
      .setLngLat([-74.006, 40.7128])
      .setPopup(new maplibregl.Popup().setHTML('<h1>New York City</h1><p>The Big Apple</p>'))
      .addTo(map);
  </script>
</body>
</html>
```

# @planisfy/typescript-config

Shared TypeScript configuration package.

Presets:

- `base.json`: shared strictness, module resolution, and language defaults.
- `source-package.json`: no-emit preset for workspace packages checked from source.
- `react-library.json`: source-package preset with React JSX transform.
- `nextjs.json`: source-package preset for Next.js apps.
- `node-service.json`: source-package preset for Node services and workers.

No package scripts.

# Planisfy UI

Shared UI components for the Planisfy platform.

> **Implementation Status**: 🟡 Package with default button component, additional components pending

---

## Overview

Reusable React components built with Radix UI and styled with Tailwind CSS.

---

## Installation

```bash
pnpm add @planisfy/ui
```

---

## Usage

```typescript
import { Button } from '@planisfy/ui/button';

export function MyComponent() {
  return <Button>Click me</Button>;
}
```

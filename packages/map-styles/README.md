# @planisfy/map-styles

Fixture MapLibre styles, source-layer contracts, sprite/glyph docs, and regional basemap build scripts.

Important data lives under `styles`, `source-layer-contract.json`, and `planetiler`.

Commands: `pnpm --filter @planisfy/map-styles test`, `lint`, `build:release`, `build:regional-release`, `build:planetiler-regional`.
